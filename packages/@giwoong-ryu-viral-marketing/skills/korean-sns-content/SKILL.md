# Korean SNS Content (한국 SNS 콘텐츠)

## Purpose
Create engaging, platform-optimized social media content for Korean audiences across Instagram, Facebook, KakaoTalk, and YouTube, maximizing reach, engagement, and conversion through culturally-relevant strategies.

## When to Use
- Planning SNS content campaigns for Korean market
- Creating platform-specific posts (Instagram, Facebook, etc.)
- Adapting global content for Korean audiences
- Increasing engagement rates on Korean SNS
- Building Korean online communities
- Launching products/services in Korea

## Core Concepts

### 1. Korean SNS Landscape

**Platform Demographics & Usage**:

| Platform | MAU (Korea) | Peak Age | Primary Use | Content Format |
|----------|-------------|----------|-------------|----------------|
| 카카오톡 | 47M (90%) | All ages | Messaging | 링크 공유, 오픈챗 |
| 인스타그램 | 17M (32%) | 20-30s | Visual | 피드, 릴스, 스토리 |
| 유튜브 | 46M (88%) | All ages | Video | 쇼츠, 장편 영상 |
| 페이스북 | 13M (25%) | 30-50s | Community | 긴 글, 링크 |
| 네이버 밴드 | 8M | 40-60s | Groups | 모임, 동호회 |

**Korean SNS Behavior**:
- **High engagement**: Comments, DMs > Public shares
- **Private sharing**: KakaoTalk 1:1, group chats
- **Visual-first**: Image quality matters more than text
- **Trendy**: Rapid trend adoption and abandonment
- **Mobile-only**: 95%+ mobile usage

### 2. Platform-Specific Strategies

#### Instagram (인스타그램)

**Content Pillars**:

**1. 피드 포스트** (Feed Posts)
```markdown
목적: 브랜드 아이덴티티, 정보 전달
최적 시간: 12PM, 6-9PM
빈도: 주 3-5회

이미지 규격:
- 정사각형: 1080x1080px (기본)
- 세로: 1080x1350px (4:5, 권장)
- 가로: 1080x566px (16:9)

Caption 길이: 200-300자
해시태그: 10-15개
```

**2. 릴스** (Reels)
```markdown
목적: 바이럴, 신규 유입
최적 시간: 6PM, 9PM
빈도: 주 3-7회

규격: 1080x1920px (9:16)
길이: 15-60초 (30초 최적)
필수 요소:
- 첫 3초 훅
- 자막 (소리 없이도 이해)
- 트렌디한 BGM
- 명확한 페이오프

해시태그: 5-10개
```

**3. 스토리** (Stories)
```markdown
목적: 일상 공유, 참여 유도
최적 시간: 8AM, 12PM, 9PM
빈도: 매일 3-5개

규격: 1080x1920px (9:16)
길이: 15초/장
수명: 24시간

인터랙션 도구:
- 투표 (찬반)
- 질문 (Q&A)
- 퀴즈
- 카운트다운
```

#### Facebook (페이스북)

**콘텐츠 특성**:
```markdown
타겟: 30-50대
최적 시간: 8-10AM, 7-9PM
빈도: 주 2-3회

콘텐츠 유형:
1. 긴 텍스트 (1,000자+)
   - 스토리텔링
   - 전문 정보
   - 의견/칼럼

2. 링크 공유
   - 썸네일 최적화 (1200x630px)
   - 메타 설명 (155자)

3. 그룹 활용
   - 커뮤니티 형성
   - 토론, 질의응답
```

#### KakaoTalk (카카오톡)

**오픈채팅 전략**:
```markdown
목적: 커뮤니티, 고객 지원

채팅방 유형:
1. 정보 공유방
   - 유용한 팁, 자료 공유
   - 스팸 금지 (신뢰 구축)

2. Q&A 방
   - 실시간 고객 문의 대응
   - 빠른 응답 (5분 이내)

3. 이벤트방
   - 한시적 프로모션
   - 선착순, 추첨

주의사항:
❌ 과도한 홍보 (퇴장당함)
✅ 가치 제공 우선 (80/20 rule)
```

### 3. Korean Tone & Style

**Tone Spectrum**:

```markdown
격식체 ←──────────────→ 반말

격식체 (합니다/입니다)
├─ 사용: 공식 발표, B2B, 금융/법률
└─ 예: "새로운 제품을 출시합니다"

준격식 (해요체)
├─ 사용: 대부분의 브랜드 SNS (가장 안전)
└─ 예: "새로운 제품을 소개해요"

반말 (해체)
├─ 사용: MZ세대 타겟, 친근한 브랜드
└─ 예: "새 제품 나왔어! 완전 대박"

주의: 반말은 신중하게 (오해의 소지)
```

**Common Korean SNS Expressions**:

| 표현 | 의미 | 사용 상황 |
|------|------|-----------|
| 혹시 몰라 저장! | Save this | 유용한 정보 공유 |
| 이거 실화? | Is this real? | 놀라운 사실/혜택 |
| 나만 알고 싶은 | My secret | 특별한 발견 공유 |
| 완전 꿀팁 | Great tip | 유용한 팁 |
| 이거 모르면 손해 | Must know | 중요한 정보 |
| ○○ 덕후라면 주목 | For ○○ fans | 특정 관심사 타겟 |

**Emoji Usage**:
```markdown
한국 SNS에서 자주 쓰이는 이모지:

✅ 체크 (완료, 동의)
❌ 엑스 (주의, 금지)
💙💛💚 컬러 하트 (강조)
😊😍🥰 웃는 얼굴 (긍정)
👍👏 손 (응원, 축하)
🔥 불 (핫한, 인기)
💡 전구 (아이디어, 팁)
📌 핀 (중요)

⚠️ 피해야 할 이모지:
🍆🍑 (성적 연상)
💀☠️ (부정적)
```

### 4. Content Formats

#### Format 1: 카드뉴스 (Card News)

**구조** (7-10 slides):
```markdown
Slide 1: 제목 + 부제목
- 큰 폰트 (80-120pt)
- 임팩트 있는 배경
- 브랜드 로고 (우측 하단)

Slide 2: 문제 제기
- "이런 고민 있으신가요?"
- 공감 유도

Slides 3-7: 솔루션 제시
- 1 slide = 1 message
- 번호 매기기 (① ② ③)
- 핵심만 간결하게

Slide 8: 요약
- "정리하면"
- 핵심 3가지 재강조

Slide 9: CTA
- "팔로우하고 더 많은 정보 받기"
- QR 코드 또는 링크

Slide 10: 출처/면책
- 정보 출처 명시
- "개인적 의견입니다" (필요시)
```

**디자인 가이드**:
```markdown
배경:
- 단색 (화이트, 파스텔)
- 그라데이션 (부드럽게)
- 이미지 + 오버레이 (가독성 확보)

폰트:
- 제목: 굵게, 60-80pt
- 본문: 보통, 40-50pt
- 충분한 행간 (1.5 이상)

색상:
- 2-3가지 색상만 사용
- 브랜드 컬러 일관성
- 대비 확보 (접근성)
```

#### Format 2: 인포그래픽

**Types**:
1. **비교형**: A vs B
2. **프로세스**: Step 1 → 2 → 3
3. **통계형**: 숫자 + 시각화
4. **타임라인**: 역사, 발전 과정
5. **체크리스트**: 항목별 체크

**Best Practices**:
```markdown
✅ 세로형 (모바일 최적)
✅ 시각적 계층 (크기로 중요도 표현)
✅ 아이콘 활용 (이해도 ↑)
✅ 데이터 출처 명시
❌ 정보 과다 (1개 주제에 집중)
❌ 글자만 가득 (시각 요소 필수)
```

#### Format 3: 짧은 영상

**15-60초 영상 구조**:
```markdown
0-3초: HOOK
- 시선 사로잡기
- 질문, 충격적 장면, 문제 제기

4-45초: CONTENT
- 핵심 메시지 전달
- 빠른 편집 (MZ세대는 짧은 컷 선호)
- 자막 필수 (소리 없이 시청)

46-60초: CTA
- 명확한 행동 유도
- 팔로우, 좋아요, 저장, 공유
```

**편집 팁**:
```markdown
트랜지션: 빠르고 자연스럽게
BGM: 트렌디한 음악 (저작권 주의)
자막:
- 위치: 중앙 또는 하단
- 크기: 큼 (모바일에서도 읽기 쉽게)
- 색상: 배경과 대비 (흰색 + 검은 테두리)
```

### 5. Engagement Tactics

**참여 유도 전략**:

**1. 질문하기**
```markdown
"여러분은 어떤 스타일 좋아하세요?"
"A vs B, 댓글로 투표해주세요!"
"이 중에 경험해본 거 있나요?"
```

**2. 공감대 형성**
```markdown
"나만 그런 줄 알았는데..."
"이거 공감 버튼 어딨어요?"
"맞아요 맞아... (좌절)"
```

**3. 비하인드 공개**
```markdown
"사실 촬영 중에..."
"실패한 버전 먼저 보여드릴게요"
"이 사진 찍으려고 ○번 도전했어요"
```

**4. 이벤트/경품**
```markdown
조건:
✅ 팔로우
✅ 좋아요 + 저장
✅ 친구 태그 2명

주의사항:
- 명확한 당첨자 발표일
- 경품 가치 명시
- 개인정보 수집 동의
```

**5. 스토리 인터랙션**
```markdown
투표: "A or B?"
질문: "궁금한 거 물어보세요"
퀴즈: "정답 맞히기 (힌트 있음)"
카운트다운: "신제품 출시까지 D-"
```

### Example 3: Facebook Long-form Post (스토리텔링)

```markdown

정말 충격을 받았습니다.

체중: 85kg (BMI 28.3, 비만)
혈압: 140/90 (고혈압 전단계)
콜레스테롤: 경계 수준

의사 선생님이 말씀하셨죠.
"지금 체중 조절 안 하시면, 약 드셔야 합니다."

그날 저녁, 거울을 보며 결심했습니다.
"이번엔 진짜 해보자."

──────────────────────────────

📌 제가 시도한 방법 (장황한 다이어트 이론 X)

1. 식단 조절 (80%)
   • 아침: 고단백 (계란, 닭가슴살)
   • 점심: 정상 식사 (회사 구내식당)
   • 저녁: 샐러드 + 과일
   • 간식: 견과류, 그릭 요거트

   ⚠️ 굶지 않았습니다! 오히려 더 자주 먹었어요.

2. 운동 (20%)
   • 주 3회, 30분 홈트레이닝
   • 출퇴근 시 계단 이용 (엘리베이터 금지)
   • 주말 산책 1시간

3. 습관 변화
   • 물 2L 이상 섭취
   • 밤 10시 이후 금식
   • 매일 아침 체중 기록

──────────────────────────────

🎯 3개월 후 결과

체중: 85kg → 75kg (-10kg)
체지방률: 28% → 21% (-7%p)
혈압: 140/90 → 125/80 (정상)

숫자보다 더 좋았던 건,
• 계단 오를 때 숨 안 참
• 옷 사이즈 100 → 95
• 만성 피로 사라짐
• 자신감 회복

저도 치킨 먹고 싶을 땐 먹었어요. (주 1회 치팅데이)

중요한 건:
✅ 지속 가능한 방법 (평생 할 수 있는)
✅ 작은 성공 경험 쌓기 (-1kg도 대단한 거)
✅ 자책하지 않기 (실패해도 다시 시작)

여러분도 할 수 있습니다!
댓글로 응원 부탁드려요. 😊
```

## Common Patterns

### Pattern 1: 시리즈 콘텐츠

```markdown

Day 1: 챌린지 소개
```

### Pattern 2: Before/After

```markdown
구조:
[Image 1] Before 상태
   └─ 문제점, 불편함 강조

[Image 2] After 상태
   └─ 개선 결과, 만족감

[Caption] 변화 스토리
   └─ 어떻게 바뀌었나, 팁 공유

적용 사례:
• 정리정돈 (지저분 → 깔끔)
• 리모델링 (낡음 → 새것)
• 다이어트 (체중 감량)
• 피부 개선 (트러블 → 맑음)
• 학습 (성적 향상)
```

### Pattern 3: 고객 후기 리포스트

```markdown
1. UGC 수집
   - 브랜드 해시태그 모니터링
   - 멘션/태그 확인
   - DM으로 허락 요청

2. 리포스트
   - 원본 유지 (크롭 최소화)
   - Credit: "@username님의 후기"
   - 감사 메시지 추가

3. 보상
   - 리포스트된 고객 혜택
   - 다음 구매 할인 쿠폰
   - 또는 소정의 선물

4. 순환
   - 주 1-2회 정기적으로
   - 다양한 고객 (공정성)
```

### Pattern 4: 트렌드 활용

```markdown
# 실시간 트렌드 모니터링

1. 트렌드 발견 (당일)
   - 인스타 탐색 탭
   - 트위터 실시간 트렌드
   - 유튜브 인기 급상승

2. 관련성 판단 (1시간)
   - 우리 브랜드와 자연스럽게 연결?
   - 긍정적 이슈인가? (부정 이슈는 패스)

3. 빠른 제작 (2-3시간)
   - 속도가 생명 (늦으면 소용없음)
   - 트렌드 해시태그 포함

4. 배포 (즉시)
   - 모든 플랫폼 동시 업로드
   - 타이밍이 가장 중요

예시:
- 날씨 이슈: "폭염에 딱 좋은 ○○"
- 드라마: "××에 나온 바로 그 제품"
- 챌린지: "△△ 챌린지 참여!"
```

## Best Practices

### ✅ DO

**콘텐츠 제작**:
- **일관성**: 포스팅 스타일, 톤, 색상 통일
- **가치 제공**: 80% 가치 제공, 20% 홍보 (80/20 Rule)
- **스토리텔링**: 단순 정보 나열보다 이야기로 전달
- **비주얼 퀄리티**: 선명하고 밝은 이미지 (어두운 사진 금물)
- **자막 필수**: 영상은 항상 자막 (소리 없이 시청)

**참여 유도**:
- **명확한 CTA**: "댓글로 알려주세요", "저장해두세요"
- **24시간 내 답변**: 댓글/DM 신속 응대
- **커뮤니티 관리**: 부정적 댓글도 정중하게 대응
- **팔로워 리스펙**: 팔로워 콘텐츠 리포스트, 언급
- **정기 이벤트**: 월 1회 경품 이벤트

**분석 & 개선**:
- **주간 리뷰**: 인사이트 데이터 확인
- **A/B 테스트**: 제목, 이미지, 시간대 테스트
- **경쟁사 벤치마크**: 업계 1위 전략 분석
- **트렌드 팔로우**: 매일 30분 SNS 트렌드 체크

### ❌ DON'T

**피해야 할 실수**:
- **무분별한 해시태그**: 무관한 인기 해시태그 30개 도배
  - 결과: 스팸 판정, 도달률 ↓

- **과도한 셀프 프로모션**: 매 포스팅마다 "구매하세요"
  - 결과: 팔로워 이탈, 언팔로우

- **저품질 이미지**: 흐릿하고 어두운 사진
  - 결과: 참여율 급감

- **일관성 없는 업로드**: 한달 침묵 → 하루 10개 업로드
  - 결과: 알고리즘 불이익

- **무시하는 댓글**: 질문/피드백에 무응답
  - 결과: 커뮤니티 약화

- **논란 조장**: 정치, 종교, 차별 관련 콘텐츠
  - 결과: 브랜드 이미지 심각한 타격

**타이밍 실수**:
- 심야 업로드 (새벽 2-6시): 참여율 최저
- 한꺼번에 여러 개: 알고리즘 분산
- 트렌드 3일 후: 이미 식은 화제

## Troubleshooting

### Issue: "참여율이 계속 떨어져요"

**원인 진단**:
1. 콘텐츠 피로도 (같은 포맷 반복)
2. 업로드 시간대 변경
3. 팔로워 관심사 변화
4. 알고리즘 변화

**Solution**:
- **콘텐츠 감사**: 최근 20개 포스팅 분석
  - 어떤 주제/포맷이 잘됐나?
  - 공통점은?
- **포맷 변화**: 3가지 유형 로테이션
  - 교육 → 엔터테인먼트 → 비하인드
- **시간대 재테스트**: 3개 시간대 A/B 테스트
- **팔로워 설문**: 스토리 투표 "어떤 콘텐츠 보고 싶어요?"

### Issue: "릴스 조회수는 높은데 팔로워가 안 늘어요"

**원인**:
- 릴스는 바이럴됐지만 프로필 방문 유도 실패
- CTA 부재 또는 약함
- 프로필 매력도 부족

**Solution**:
1. **릴스 마지막 화면**: 팔로우 버튼 명확히
   ```
   "더 많은 팁 보려면
    팔로우 👆 (손가락 포인팅)"
   ```

2. **프로필 최적화**:
   - 바이오: 명확한 가치 제안
   - 하이라이트: 베스트 콘텐츠 정리
   - 고정 포스팅: 대표 콘텐츠

3. **시리즈 콘텐츠**: "Part 1" → "Part 2는 피드에"
   - 연속성으로 프로필 방문 유도

### Issue: "악성 댓글이 달렸어요"

**대응 프로토콜**:

**1. 평가 (즉시)**:
- 단순 비판인가? → 정중하게 답변
- 악의적 비방인가? → 삭제 고려
- 허위 사실인가? → 반박 + 증거 제시

**2. 대응 (2시간 이내)**:
```markdown
단순 비판:
"소중한 의견 감사합니다.
 ○○ 부분은 저희도 개선 중입니다."

악의적 비방:
"건설적 비판은 환영하지만,
 근거 없는 비방은 삭제하겠습니다."
→ 댓글 삭제 + 차단

허위 사실:
"사실과 다른 내용입니다.
 [증거/링크] 확인 부탁드립니다."
```

**3. 예방**:
- 커뮤니티 가이드라인 명시
- "존중하는 댓글 문화" 강조
- 심한 경우 법적 대응 경고

### Issue: "해시태그를 많이 써도 도달률이 낮아요"

**원인**:
- 부적절한 해시태그 선택
  - 너무 대형: #사랑 (경쟁 과다)
  - 너무 소형: #내방책상정리 (검색 없음)
  - 무관한 태그: 스팸 판정

**Solution**:
1. **해시태그 분석 도구 사용**:
   - Instagram Insights
   - 각 해시태그별 도달 수 확인

2. **3-Tier 전략**:
   ```
   대형 (100K+ posts): 2-3개
   중형 (10K-100K): 7-10개
   소형 (1K-10K): 3-5개
   ```

3. **관련성 최우선**:
   - 본문에 실제로 언급된 키워드만
   - 이미지 내용과 직접 관련

4. **브랜드 해시태그**:
   - 자체 해시태그 만들기
   - 캠페인별 고유 태그
