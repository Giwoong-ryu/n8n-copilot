# Korean Grammar Checker (한국어 문법 검사)

## Purpose
Quick reference guide for the most common Korean grammar mistakes, providing fast proofreading and validation for Korean content creators.

## When to Use
- Quick grammar checks before publishing
- Validating common error patterns
- Last-minute proofreading
- Checking frequently confused words
- Verifying particle usage
- Confirming proper spacing

## Core Concepts

### 1. Top 20 Common Mistakes (빈출 오류 TOP 20)

**1. 되/돼 혼동**
```markdown
Rule: "되어"를 줄인 것 = "돼"

✅ 안 돼 (= 안 되어)
❌ 안 되

✅ 됐어 (= 되었어)
❌ 됬어

✅ 되다 (원형)
❌ 돼다

빠른 체크: "되어"로 바꿔보기
"안 되어" → 맞음 → "안 돼" ✅
```

**2. 어떻게/어떡해 혼동**
```markdown
Rule:
• 어떻게 = How (부사)
• 어떡해 = 어떻게 해 (줄임말)

✅ 이거 어떻게 해? (How to do?)
✅ 이거 어떡해? (What to do?)
❌ 이거 어떻해?

✅ 어떻게 하면 돼요?
❌ 어떡해 하면 돼요?
```

**3. 웬/왠 혼동**
```markdown
Rule:
• 웬 = "어찌된" (관형사)
• 왠지 = "무슨 까닭인지" (부사)

✅ 웬일이야?
❌ 왠일이야?

✅ 왠지 모르겠어
❌ 웬지 모르겠어

✅ 웬만하면
❌ 왠만하면
```

**4. 띄어쓰기: 의존명사**
```markdown
항상 띄어쓰기:

✅ 할 수 있다
❌ 할수있다

✅ 먹을 것이다
❌ 먹을것이다

✅ 만난 지 3년
❌ 만난지 3년

✅ 5개월 만에
❌ 5개월만에

✅ 한 번만
❌ 한번만
```

**5. 띄어쓰기: 부정 부사**
```markdown
"안", "못" 띄어쓰기:

✅ 안 돼
❌ 안돼

✅ 못 해
❌ 못해

✅ 안 갔어
❌ 안갔어

예외: 못생기다, 못되다 (붙여쓰기)
```

**6. -르 불규칙 활용**
```markdown
Rule: 르 → 라/러

빠르다:
✅ 빨라요
❌ 빠라요

부르다:
✅ 불러요
❌ 부러요

다르다:
✅ 달라요
❌ 다라요
```

**7. 가르치다/가리키다**
```markdown
• 가르치다 = teach
✅ 선생님이 가르쳐요
❌ 선생님이 가르켜요

• 가리키다 = point at
✅ 손으로 가리켜요
❌ 손으로 가르켜요
```

**8. 틀리다/다르다**
```markdown
• 틀리다 = incorrect (오답)
✅ 답이 틀려요

• 다르다 = different (차이)
✅ 나는 너와 달라

❌ 나는 너와 틀려 (wrong!)
```

**9. 로서/로써**
```markdown
• 로서 = as (자격, 신분)
✅ 부모로서

• 로써 = with (도구, 수단)
✅ 연필로써 쓰다

받침 O → 으로서/으로써
✅ 친구로서
✅ 칼로써
```

**10. -게/게**
```markdown
• -게 (부사형 어미)
✅ 빠르게 달려요

• -게 (의도)
✅ 먹게 해주세요

❌ 빠르게요 (X)
✅ 빨라요 (O) or 빠릅니다 (O)
```

**11. 낫다/낳다/났다**
```markdown
• 낫다 = better, heal
✅ 감기가 나았어요

• 낳다 = give birth
✅ 아기를 낳았어요

• 났다 = 나다 past
✅ 불이 났어요
```

**12. 이따가/있다가**
```markdown
• 이따가 = later (시간)
✅ 이따가 봐요

• 있다가 = exist + then
✅ 집에 있다가 나갈게요

❌ 있따가 (X)
```

**13. 던지/든지**
```markdown
• -던지 (과거 회상)
✅ 먹던지 말든지 (과거)

• -든지 (선택)
✅ 먹든지 말든지 (현재 선택)

✅ 뭐든지 좋아요
❌ 뭐던지 좋아요
```

**14. 안/않**
```markdown
• 안 = not (부정 부사, 띄어쓰기)
✅ 안 가요

• 않- = not (부정 어미, 붙여쓰기)
✅ 가지 않아요

❌ 않 가요 (X)
❌ 가지안아요 (X)
```

**15. 데/대**
```markdown
• -ㄴ/은/는 데 = 것 (의존명사)
✅ 먹는 데 시간이 걸려요
(띄어쓰기!)

• -ㄴ/은/는대 = ~ㄴ/은/는다고 해
✅ 맛있는대? (맛있다고 해?)
(붙여쓰기!)

✅ 가는 데 (장소)
✅ 가는데 (역접)
```

**16. -ㅂ니다/-습니다**
```markdown
Rule:
받침 X → -ㅂ니다
받침 O → -습니다

✅ 가ㅂ니다
✅ 먹습니다

❌ 가습니다 (X)
❌ 먹ㅂ니다 (X)
```

**17. -ㄹ게요/-을게요**
```markdown
Rule:
받침 X → -ㄹ게요
받침 O → -을게요

✅ 할게요 (하다)
✅ 먹을게요 (먹다)

❌ 하을게요 (X)
❌ 먹ㄹ게요 (X)
```

**18. 사귀다/사괴다**
```markdown
✅ 사귀다 (correct)
✅ 사귀어요, 사귀었어요

❌ 사괴다 (X)
❌ 사괴요 (X)

친구를 사귀다 = make friends
연애를 사귀다 = date
```

**19. 개/게**
```markdown
• 개 = 수량
✅ 사과 한 개
(띄어쓰기!)

• -게 = 어미
✅ 빨리 가게

❌ 사과 한게 (X)
```

**20. -네요/-내요**
```markdown
✅ -네요 (correct)
✅ 좋네요, 예쁘네요

❌ -내요 (X)
❌ 좋내요 (X)

감탄, 발견: -네요
```

### 2. Particle Quick Reference (조사 빠른 참조)

**주격 조사 (Subject)**
```markdown
이/가:
• 받침 O → 이
  철수이 (X) → 철수가 (O)

• 받침 X → 가
  나가 (O)
```

**목적격 조사 (Object)**
```markdown
을/를:
• 받침 O → 을
  밥을 먹다 ✅

• 받침 X → 를
  사과를 먹다 ✅
```

**처소격 조사 (Location)**
```markdown
에/에서:
• 에 = 목적지, 시간
  학교에 가다 ✅
  3시에 만나다 ✅

• 에서 = 장소 (동작)
  학교에서 공부하다 ✅

Common Error:
❌ 학교를 가다 (X)
✅ 학교에 가다 (O)
```

### 3. Speed Check Patterns

**Pattern 1: 3초 체크 (의존명사)**
```markdown
Q: 띄어쓰기 맞나요?

"할수있다" → "할 수 있다" ✅
"먹을것이다" → "먹을 것이다" ✅
"만난지" → "만난 지" ✅

Rule: 수, 것, 지, 만, 번 → 항상 띄어쓰기
```

**Pattern 2: 5초 체크 (되/돼)**
```markdown
Q: 되? 돼?

Step 1: "되어"로 바꿔보기
"안 되어" → 맞음 → "안 돼" ✅
"되었다" → 맞음 → "됐다" ✅

Step 2: "되다"의 활용형이면 붙여쓰기
"될 것이다" ✅
"되려면" ✅
```

**Pattern 3: 10초 체크 (전체)**
```markdown
1. 의존명사 띄어쓰기 (수, 것, 지, 만, 번)
2. 부정 부사 띄어쓰기 (안, 못)
3. 되/돼 확인
4. 어떻게/어떡해 확인
5. 조사 확인 (을/를, 이/가)
```

## Examples

### Example 1: 블로그 포스팅 빠른 검사

**Before**:
```
오늘 맛집에 갔다. 음식이 정말 맛있었다.
친구들과 같이갔는데, 다들 만족했다.
다음에 또갈것같다. 추천한다.

Errors: 6개
```

**After** (교정 + 설명):
```
오늘 맛집에 갔다. 음식이 정말 맛있었다.
친구들과 같이 갔는데, 다들 만족했다.
         ↑ 띄어쓰기 (부사)
다음에 또 갈 것 같다. 추천한다.
         ↑   ↑ 띄어쓰기 (것 = 의존명사)

교정 시간: 30초
```

### Example 2: SNS 포스팅 검사

**Before**:
```
오늘 날씨 진짜 좋네요!
나도 나가고싶다 ㅠㅠ
집에만 있는데 심심해요

```

**After**:
```
오늘 날씨 진짜 좋네요!

**Before**:

Errors: 3개
```

**After**:
```

다음 주 월요일에 회의가 있을 것 같습니다.

    ↑            ↑
  띄어쓰기      띄어쓰기

교정 시간: 20초
```

## Common Patterns

### Pattern 1: Error Severity Levels

```markdown
Level 1 - Critical (치명적 오류):
독자가 의미를 오해할 수 있음
• 틀리다/다르다 혼동
• 낫다/낳다 혼동
• 가르치다/가리키다 혼동

→ 즉시 수정 필수

---

Level 2 - High (높은 오류):
전문성이 의심받을 수 있음
• 되/돼 혼동
• 의존명사 띄어쓰기
• 어떻게/어떡해 혼동

→ 공식 문서, 블로그는 반드시 수정

---

Level 3 - Medium (중간 오류):
눈에 띄지만 의미는 통함
• 부정 부사 띄어쓰기 (안, 못)
• 로서/로써 혼동

→ 시간 있으면 수정

---

Level 4 - Low (낮은 오류):
일상 대화에서 흔히 틀림
• -네요/-내요
• 사귀다/사괴다

→ SNS는 OK, 공식 문서는 수정
```

### Pattern 2: Platform-Based Checking

```markdown
공식 문서 (비즈니스, 법률):
✅ 모든 레벨 오류 수정
✅ 격식체 통일
✅ 띄어쓰기 100% 준수
검사 시간: 5분

블로그 포스팅:
✅ Level 1-2 오류 수정
✅ 의존명사 띄어쓰기
⚠️ Level 3-4는 선택
검사 시간: 2분

SNS (인스타, 트위터):
✅ Level 1 오류만 수정
⚠️ 띄어쓰기 어느정도 허용
⚠️ 구어체 표현 OK
검사 시간: 30초

친구 메시지:
⚠️ 의미만 통하면 OK
검사 시간: 0초 (안 해도 됨)
```

## Best Practices

### ✅ DO

- **자주 틀리는 것 암기**: TOP 20 외우기
- **맞춤법 검사기 활용**: 네이버, 부산대
- **소리내어 읽기**: 어색한 부분 발견
- **의심되면 검색**: 확신 없으면 찾아보기
- **개인 오류 노트**: 자주 틀리는 것 정리

### ❌ DON'T

- **검사기 맹신**: AI도 틀릴 수 있음
- **과도한 수정**: 구어체 허용 범위 이해
- **시간 낭비**: 플랫폼별 검사 수준 조절
- **맥락 무시**: 의미 통하면 OK인 경우도 있음

## Troubleshooting

### Issue: "맞춤법 검사기가 틀렸다고 하는데..."

**Solution**:
- 신조어는 검사기가 인식 못함 ("갓생", "킹받네")
- 구어체 표현도 오류로 표시 ("그쵸", "맞쥬")
- 타겟과 플랫폼 고려해서 최종 판단

### Issue: "너무 많은 오류가 나와요"

**Solution**:
```markdown
우선순위 수정:
1순위: Level 1 (치명적)
2순위: Level 2 (높음)
3순위: 의존명사 띄어쓰기
4순위: 나머지

시간 없으면 1-2순위만 수정
```

### Issue: "어떤 맞춤법 검사기를 써야 하나요?"

**추천 도구**:
```markdown
1. 네이버 맞춤법 검사기
   • 무료
   • 한국어 특화
   • URL: https://search.naver.com/search.naver?where=nexearch&query=맞춤법검사기

2. 부산대 맞춤법 검사기
   • 무료
   • 상세 설명
   • URL: http://speller.cs.pusan.ac.kr/

3. 한국어 맞춤법/문법 검사기
   • Chrome Extension
   • 실시간 체크

사용 순서:
네이버 검사 → 부산대 재검사 → 수동 최종 확인
```
