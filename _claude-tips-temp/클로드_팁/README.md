## 🎯 스킬셋 (11개)

Claude Code를 위한 강력한 AI 자동화 스킬셋입니다.

### 카테고리별 스킬

#### 개발 자동화 (5개)
- **code-changelog**: 코드 변경사항 자동 문서화
- **meta-prompt-generator**: 커스텀 슬래시 커맨드 생성
- **prompt-enhancer**: 요청을 상세 요구사항으로 변환
- **codex-claude-loop**: 이중 AI 검증 시스템
- **codex**: OpenAI Codex 코드 분석

#### 프로젝트 초기화 (2개)
- **flutter-init**: Flutter Clean Architecture 프로젝트
- **nextjs15-init**: Next.js 15 App Router 프로젝트

#### 콘텐츠 생성 (4개)
- **landing-page-guide**: 랜딩페이지 제작 가이드
- **card-news-generator**: 카드 뉴스 자동 생성
- **card-news-generator-v2**: 배경 이미지 지원 버전
- **midjourney-cardnews-bg**: Midjourney 프롬프트 생성

**👉 [스킬셋 상세 문서 보기](./스킬셋/README.md)**
