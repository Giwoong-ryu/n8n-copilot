## 📖 주요 내용

### 1. Claude 도구 완벽 이해
- **Projects** (LEVEL 0) - 작업 패턴 저장하고 재사용
- **Commands** (LEVEL 1) - 슬래시 명령어로 반복 작업 자동화
- **Agents** (LEVEL 2) - 복잡한 작업 자동 실행
- **Skills** (LEVEL 3) - 전문 지식 자동 적용
- **MCP** - 외부 데이터 실시간 연결

### 2. 37개 스킬 가이드
```
🔧 n8n 워크플로우 (6개)
🎨 디자인 & 프론트엔드 (5개)
📄 문서 처리 (4개)
🚀 마케팅 (2개)
💡 생산성 (4개)
🛠️ 개발 도구 (6개)
🪟 Windows 전용 (10개)
```

### 3. 토큰 사용량 완전 분석
- 도구별 토큰 사용량
- 실제 작업별 예시 (보고서, 발표자료, 이메일 등)
- 도구 사용 전후 비교

### 4. 요금제 가이드
- Free / Pro / Max 상세 비교
- 작업량에 따른 추천 플랜
- 비용 대비 효율 분석
