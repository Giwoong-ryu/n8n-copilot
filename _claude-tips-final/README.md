## 📖 가이드 내용

### 1️⃣ 메인 가이드 (클로드_팁.html)
Claude의 핵심 기능을 일상 용어로 설명합니다:
- **Projects** (작업공간) - 전문가 모드 설정하기
- **Commands** (바로가기) - 자주 쓰는 명령 저장하기
- **Agents** (자동 작업자) - 복잡한 작업 자동화
- **Skills** (전문 능력) - 특화된 기능 추가
- **MCP** (연결 도구) - 외부 도구와 연동

### 2️⃣ 스킬셋 가이드 (클로드_팁_스킬셋.html)
37개의 실전 스킬을 카테고리별로 정리:
- **n8n 워크플로우** (6개) - 자동화 전문 스킬
- **디자인 & 프론트엔드** (5개) - UI/UX 개발 스킬
- **문서 처리** (4개) - 문서 작업 효율화
- **마케팅** (2개) - 콘텐츠 마케팅 스킬
- **생산성** (4개) - 개발 생산성 향상
- **개발 도구** (6개) - 코드 품질 개선
- **Windows 전용** (10개) - Windows 환경 최적화

### 3️⃣ 관련 프로젝트
- **[my-skills](https://github.com/Giwoong-ryu/my-skills)** - Windows 전용 스킬 실제 코드
- **[n8n-copilot](https://github.com/Giwoong-ryu/n8n-copilot)** - n8n 워크플로우 자동화 도구

## 💡 활용 방법
