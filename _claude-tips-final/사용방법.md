## 🔧 문제 해결

### Q: 404 오류가 계속 나요
**A**: GitHub Pages가 업데이트되려면 1-2분 소요됩니다. 잠시 후 새로고침하세요.

### Q: 여전히 포트폴리오 페이지가 나와요
**A**: 브라우저 캐시를 지우고 다시 시도하세요 (Ctrl + Shift + R)

### Q: GitHub Actions가 작동하지 않아요
**A**: 레포지토리 Settings > Pages에서 "Source"를 "GitHub Actions"로 설정하세요.
