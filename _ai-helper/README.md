## 🚀 로컬 개발

```bash
# 레포지토리 클론
git clone https://github.com/Giwoong-ryu/ai-helper.git

# 폴더 이동
cd ai-helper

# 로컬 서버 실행 (Python 3)
python -m http.server 8000

# 브라우저에서 열기
open http://localhost:8000
```
