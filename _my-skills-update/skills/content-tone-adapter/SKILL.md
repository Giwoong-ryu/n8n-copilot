# Content Tone Adapter (콘텐츠 톤 변환)

## Purpose
Transform Korean content tone and style to match different target audiences, platforms, and contexts while maintaining the core message and ensuring natural, culturally-appropriate language.

## When to Use
- Adapting formal content for younger, casual audiences
- Converting casual blog posts to professional business documents
- Adjusting content tone for different age groups (10s, 20s, 30s+)
- Repurposing content across platforms (LinkedIn → Instagram)
- Creating multiple versions of the same content for A/B testing
- Matching brand voice guidelines across different channels

## Core Concepts

### 1. Korean Tone Spectrum (격식 스펙트럼)

**5-Level Formality System**:

| Level | 종결어미 | 사용 상황 | 타겟 |
|-------|---------|----------|------|
| **격식체 (Formal)** | ~합니다/~입니다 | 비즈니스, 뉴스, 학술 | 30대+ 전문직 |
| **준격식체 (Semi-formal)** | ~해요/~이에요 | 블로그, 온라인 강의 | 전 연령, 일반 독자 |
| **구어체 (Conversational)** | ~하죠/~이죠 | 친근한 블로그, 유튜브 | 20-30대 |
| **친근체 (Friendly)** | ~해/~해요 혼용 | SNS, 인스타그램 | 10-30대 |
| **반말 (Casual)** | ~해/~이야 | 친구, 밈, 유머 콘텐츠 | 10-20대 |

**Tone Characteristics**:

```markdown
격식체 (Formal):
• 완전한 문장 구조
• 한자어 사용 (예: 변경 → 바꿈, 진행 → 나아감)
• 수동태 허용 ("~됩니다", "~입니다")
• 객관적, 거리감 있음

예시: "본 서비스는 고객님의 편의를 위해 개선되었습니다."

---

준격식체 (Semi-formal):
• 부드러운 격식
• 한글+한자어 혼용
• 능동태 선호
• 친절하지만 전문적

예시: "서비스를 더 편리하게 사용하실 수 있도록 개선했어요."

---

구어체 (Conversational):
• 입말 느낌
• 한글 위주
• 축약형 사용 ("그쵸", "그렇죠")
• 공감 표현 ("~잖아요", "~니까요")

예시: "서비스가 훨씬 편해졌죠? 이제 클릭 한 번이면 끝이에요!"

---

친근체 (Friendly):
• 이모지 사용
• 줄임말 허용 ("완전", "진짜")
• 감정 표현 강함
• 질문/대화 형식

예시: "서비스 완전 편해졌어요! 😍 이제 클릭 한 번이면 끝~"

---

반말 (Casual):
• 완전한 구어체
• 신조어 사용 ("갓생", "꾸안꾸")
• 축약 과감 ("그렇게→그케", "이렇게→이케")
• 친구 대화 느낌

예시: "서비스 완전 편해짐 ㅋㅋ 이제 클릭 ㄱㄱ"
```

### 2. Audience Analysis (타겟 분석)

**Age-based Tone Guide**:

**10대 (Teens)**:
```markdown
언어 특징:
• 신조어 적극 활용: "갓생", "킹받네", "레전드"
• 이모티콘 과다: "ㅋㅋㅋ", "ㅠㅠ", "ㅇㅈ"
• 줄임말: "ㅇㅋ", "ㄱㅅ", "ㅈㅅ"
• 영어 섞어 쓰기: "완전 GOAT", "진짜 real"

예시:
Before: "이 제품은 정말 좋습니다."
After: "이거 진짜 레전드 ㅋㅋㅋ 완전 대박이야"
```

**20대 (20s)**:
```markdown
언어 특징:
• 신조어 + 표준어 혼용
• 적절한 이모지 사용
• 유행어 빠르게 수용
• SNS 용어 친숙: "인싸", "아싸", "TMI"

예시:
Before: "이 제품은 사용이 간편합니다."
After: "사용법 진짜 쉬움 👍 초보도 가능할듯"
```

**30대 (30s)**:
```markdown
언어 특징:
• 준격식체 선호
• 신조어 제한적 사용
• 이모지 절제
• 정보 중심

예시:
Before: "이 제품은 효과적입니다."
After: "사용해보니 확실히 효과가 있네요. 추천합니다!"
```

**40대+ (40+)**:
```markdown
언어 특징:
• 격식체 선호
• 신조어 거의 없음
• 이모지 최소
• 정확한 표현 중시

예시:
Before: "이거 완전 좋음"
After: "이 제품은 품질이 우수하여 만족스럽습니다."
```

### 3. Platform-Specific Tones

**네이버 블로그 (Naver Blog)**:
```markdown
톤: 준격식체 + 친근함
특징:
• 해요체 기본
• 적절한 이모지 (😊 👍 💡)
• 정보 제공 중심
• 신뢰감 중요

구조:
[인사] → [문제 제기] → [정보 제공] → [요약] → [이웃추가 요청]

예시:
"안녕하세요! 오늘은 ○○에 대해 알려드릴게요.
혹시 이런 고민 있으셨나요?

[본문...]

도움이 되셨다면 이웃추가 부탁드려요! 😊"
```

**인스타그램 (Instagram)**:
```markdown
톤: 친근체 + 감성
특징:
• 짧은 문장 (15자 이내)
• 줄바꿈 자주 (3줄마다)
• 이모지 많이 (문장마다 1개)
• 해시태그 중요

구조:
[훅] → [감정 표현] → [정보] → [CTA] → [해시태그]

예시:
"오늘의 루틴 ☕️

아침 일찍 일어나서
요가하고
맛있는 브런치까지 🥐

완벽한 하루의 시작이었어요 ✨

여러분도 시도해보세요!

#모닝루틴 #갓생살기"
```

**유튜브 커뮤니티 (YouTube Community)**:
```markdown
톤: 구어체 + 친근함
특징:
• 말하듯이 쓰기
• 물음표 활용 (참여 유도)
• 이모지 적당히
• 댓글 유도

예시:

```

**LinkedIn (링크드인)**:
```markdown
톤: 격식체 + 전문성
특징:
• 합니다체
• 한자어 사용
• 이모지 최소
• 전문 용어 OK

예시:

```

### 4. Tone Conversion Techniques

**Technique 1: Formality Adjustment (격식도 조정)**

```markdown
Formal → Casual 변환 규칙:

1. 종결어미 변경:
   ~합니다 → ~해요 → ~해

2. 한자어 → 순우리말:
   변경 → 바꿈
   진행 → 나아감
   완료 → 끝남

3. 수동태 → 능동태:
   개선되었습니다 → 개선했어요

4. 문장 단순화:
   본 서비스는 고객님의 편의를 위하여 개선되었습니다.
   → 서비스를 더 편하게 쓸 수 있게 바꿨어요!

5. 감정 추가:
   만족스럽습니다 → 정말 좋아요! / 완전 만족해요!
```

**Technique 2: Emotional Intensity (감정 강도 조절)**

```markdown
중립 → 긍정 강화:

Level 1 (약한 긍정):
"좋습니다" → "괜찮아요"

Level 2 (보통 긍정):
"좋습니다" → "좋아요" / "마음에 들어요"

Level 3 (강한 긍정):
"좋습니다" → "정말 좋아요!" / "너무 만족해요!"

Level 4 (최강 긍정):
"좋습니다" → "완전 대박! 진짜 최고예요! 😍"

이모지 활용:
Level 1: 이모지 없음
Level 2: 😊 👍 (기본)
Level 3: 😍 💕 🎉 (강조)
Level 4: 🔥🔥🔥 OMG 😱💯 (과장)
```

**Technique 3: Sentence Structure Transformation**

```markdown
긴 문장 → 짧은 문장 (SNS용):

Before (격식체, 53자):
"이 제품은 사용이 간편하고 디자인도 세련되어 많은 분들께 추천드립니다."

After (친근체, 3 문장):
"사용법 진짜 쉬워요!
디자인도 완전 예뻐요 ✨
꼭 써보세요!"

변환 규칙:
1. 접속사 기준으로 분리 (그리고, ~하고)
2. 각 문장 15자 이내
3. 감정 표현 강화
4. 이모지 추가
```

## Examples

### Example 1: 비즈니스 → 블로그 (Business to Blog)

**Before (격식체, LinkedIn)**:
```

데이터 기반 의사결정 체계를 구축하기 위함입니다.

주요 성과:
- 캠페인 운영 시간 60% 단축
- 타겟팅 정확도 45% 향상
- ROI 230% 증가

향후 계획:
추가 AI 도구 도입을 통해 전사적 디지털 전환을
가속화할 예정입니다.
```

**After (준격식체, Naver Blog)**:
```

대해 공유해보려고 해요 💡

📍 왜 도입했나요?

데이터 분석도 엑셀로 수동으로 했거든요 😅

1️⃣ 시간 절약 (60% 단축!)

2️⃣ 타겟팅 정확도 UP
AI가 자동으로 최적 타겟을 찾아줘서
광고 효율이 45%나 올랐어요 🎯

3️⃣ ROI 2배 이상 증가
투자 대비 수익이 230%까지 올라갔어요!

[이미지: 성과 그래프]

💬 솔직한 장단점

• 자동화로 시간 많이 절약됨

### Example 3: 논문 → 유튜브 (Academic to YouTube)

**Before (격식체, 학술)**:
```
제목: 한국 청소년의 SNS 사용 패턴과 정신건강의 상관관계

초록:
본 연구는 만 13-18세 청소년 1,200명을 대상으로
SNS 사용 시간과 우울증 지수 간의 관계를 분석하였다.

연구 결과, 일평균 SNS 사용 시간이 4시간을
초과하는 집단에서 우울증 유병률이 유의미하게
높게 나타났다 (p<0.01).

특히 타인과의 비교 행동이 많을수록 자존감 저하가
심화되는 것으로 확인되었다.

결론적으로, 청소년의 건강한 SNS 사용을 위한
교육 프로그램 개발이 필요한 것으로 판단된다.
```

**After (구어체, YouTube Description)**:
```

오늘은 청소년 SNS 사용과 정신건강에 관한
충격적인 연구 결과를 가져왔어요 😮

🔬 연구 개요
• 대상: 중고등학생 1,200명
• 기간: 6개월
• 주제: SNS vs 우울증 상관관계

📊 핵심 발견

우울증 걸릴 확률이 2배나 높았대요!

특히 이런 행동 위험:

❌ 인플루언서 따라하기

→ 자존감이 떨어지고 우울해짐 😔

연구진들이 제안한 건강한 SNS 사용법:

1️⃣ 하루 2시간 이내로 제한
2️⃣ 비교하지 말기 (내 삶은 내 것!)
3️⃣ 자기 전 1시간은 폰 금지

## Common Patterns

### Pattern 1: Quick Tone Conversion Checklist

```markdown
Step 1: 타겟 확인 (5초)
□ 나이: 10대 / 20대 / 30대 / 40+
□ 플랫폼: 블로그 / 인스타 / 유튜브 / 링크드인
□ 목적: 정보 / 홍보 / 소통 / 교육

Step 2: 종결어미 변경 (30초)
□ 격식체 → 해요체 / 반말
□ 일관성 유지

Step 3: 단어 치환 (1분)
□ 한자어 → 순우리말 (캐주얼용)
□ 전문용어 → 쉬운 말

Step 4: 문장 구조 조정 (2분)
□ 긴 문장 → 짧은 문장 (SNS용)
□ 수동태 → 능동태

Step 5: 감정/이모지 추가 (1분)
□ 타겟별 적절한 이모지
□ 감정 표현 강도 조절

Step 6: 최종 검토 (30초)
□ 소리 내어 읽어보기
□ 타겟 입장에서 자연스러운지 확인
```

### Pattern 2: A/B Testing Versions

```markdown
동일 메시지, 3가지 톤으로 변환:

원본 메시지:
"신제품 출시 기념 30% 할인"

Version A (격식체 - 40대+):
"신제품 출시를 기념하여 30% 특별 할인 혜택을
드립니다. 이번 기회를 놓치지 마세요."

Version B (준격식체 - 30대):
"신제품 나왔어요! 🎉
출시 기념으로 30% 할인해드려요.
지금 바로 확인해보세요!"

Version C (친근체 - 20대):
"신상 떴다! 🔥
출시 기념 30% 할인 ㄷㄷ
놓치면 후회함 ㅠㅠ
지금 바로 겟 👉👉"

테스트 결과:
• Version A: CTR 2.3% (40대+ 타겟)
• Version B: CTR 4.7% (30대 타겟)
• Version C: CTR 8.1% (20대 타겟)
```

### Pattern 3: Brand Voice Template

```markdown
브랜드별 톤 가이드라인 예시:

Brand: 건강식품 스타트업
Target: 20-30대 직장인
Platforms: 인스타, 블로그

✅ DO:
• 해요체 사용
• 건강한 라이프스타일 강조
• 따뜻한 감성 ("○○님", "함께")
• 정보 제공 + 공감
• 이모지: 🌿 💚 ✨ 🥗

예시:
"건강한 아침을 시작하는 방법, 함께 알아봐요! 🌿
바쁜 아침에도 챙길 수 있는 간편한 팁이에요."

❌ DON'T:
• 반말 사용
• 과도한 신조어 ("갓생", "킹받네")
• 공격적 어조
• 과장 ("대박", "완전 미쳤다")
• 이모지: 🔥 💯 😱
```

## Best Practices

### ✅ DO

**변환 전 준비**:
- **원문 의도 파악**: 핵심 메시지 먼저 이해
- **타겟 페르소나 설정**: 구체적일수록 좋음
- **플랫폼 특성 고려**: 글자수 제한, 이미지 비중
- **브랜드 가이드 확인**: 일관성 유지

**변환 중**:
- **자연스러움 우선**: 억지로 끼워 맞추지 않기
- **문화적 맥락 고려**: 한국어만의 뉘앙스 살리기
- **일관성 유지**: 한 콘텐츠 내 톤 통일
- **핵심 메시지 보존**: 톤 바뀌어도 내용은 동일

**변환 후**:
- **소리 내어 읽기**: 어색한 부분 발견
- **타겟 반응 예측**: 실제로 쓰는 표현인지 확인
- **A/B 테스트**: 여러 버전 비교
- **피드백 수집**: 실제 타겟 의견 청취

### ❌ DON'T

**절대 금지**:
- **의미 왜곡**: 톤 바꾸려다 내용 바뀌면 안 됨
- **과도한 변환**: 원본과 완전히 다른 글로 변질
- **타겟 무시**: 40대에게 "ㅋㅋㅋ" 쓰지 말기
- **문화적 부적절**: 세대 간 오해 유발 표현 피하기
- **브랜드 정체성 손실**: 일관성 없으면 혼란

**피해야 할 실수**:
- 신조어 남발 (타겟이 모르면 무의미)
- 이모지 과다 (가독성 저하)
- 톤 혼재 (격식체 + 반말 섞지 말기)
- 번역기 느낌 (부자연스러운 표현)
- 맥락 없는 유행어 (억지스러움)

## Troubleshooting

### Issue: "변환했는데 어색해요"

**원인**:
1. 종결어미만 바꾸고 어휘는 그대로
2. 문장 구조가 여전히 격식적
3. 감정 표현 부족

**Solution**:
```markdown
Step 1: 단어 수준 변환
Before: "본 제품은 품질이 우수합니다."
→ "이 제품은 품질이 좋아요."

Step 2: 문장 구조 변환
→ "품질 정말 좋아요."

Step 3: 감정 추가
→ "품질 정말 좋아요! 써보니 확실히 다르더라고요."

Step 4: 타겟 언어 추가
→ "품질 진짜 좋음! 써보니 확실히 다름 👍"
```

### Issue: "타겟 연령대 언어를 모르겠어요"

**Solution: 리서치 방법**
```markdown
10대 언어 배우기:
1. 유튜브 쇼츠 / 틱톡 댓글 읽기
2. 인기 유튜버 영상 시청
3. 인스타그램 인기 게시물 분석
4. 네이버 카페 (게임, 아이돌) 방문

20대 언어 배우기:
1. 에브리타임 게시판
2. 인스타그램 밈 계정
3. 유튜브 브이로그
4. 트위터 트렌드

30대 언어 배우기:
1. 네이버 블로그 인기글
2. 맘카페, 육아 커뮤니티
3. LinkedIn 게시물
4. 직장인 커뮤니티

실전 연습:
• 매일 10분씩 타겟 플랫폼 둘러보기
• 인기 게시물 3개 분석
• 주요 표현 노트 정리
• 직접 써보고 피드백 받기
```

### Issue: "브랜드 톤과 타겟 톤이 안 맞아요"

**Case Study**:
```markdown
상황: 법률 서비스 (보수적 브랜드) → 20대 타겟

잘못된 접근:
"법률 상담 필요하면 DM 고고~~ ㅋㅋ
변호사 쌤들 완전 프로임 💯"
→ 브랜드 신뢰도 추락

올바른 접근:
"법률 상담이 필요하신가요?
20-30대를 위한 쉬운 법률 가이드를 제공합니다.

✅ 쉬운 용어로 설명
✅ 무료 초기 상담
✅ 카톡 상담 가능

어려운 법률, 쉽게 풀어드릴게요."

해결 원칙:
• 브랜드 정체성: 유지 (전문성, 신뢰)
• 언어 스타일: 조정 (쉬운 설명, 접근성)
• 플랫폼 적응: 카톡/DM 제공
• 균형: 전문적이면서도 친근하게
```
