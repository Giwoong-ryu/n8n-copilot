# Thumbnail Copywriting (썸네일 카피라이팅)

## Purpose
Create compelling, high-CTR thumbnail copy for Korean YouTube videos, blog posts, and social media content using proven click-triggering techniques and optimal text design.

## When to Use
- Designing YouTube video thumbnails
- Creating blog post header images
- Making Instagram/Facebook cover images
- Designing course/webinar promotion images
- A/B testing different thumbnail versions
- Optimizing low-performing content CTR

## Core Concepts

### 1. Korean Thumbnail Text Principles

**The 3-Second Rule**:
```markdown
썸네일 텍스트는 3초 안에 읽혀야 함

✅ Good (3초 이내):
"월 1000만원 버는 법"
→ 9자, 명확한 메시지

❌ Bad (3초 초과):
"부업으로 시작해서 월 1000만원까지 번 나만의 노하우"
→ 27자, 너무 김
```

**Optimal Character Count**:

| 플랫폼 | 권장 글자수 | 최대 글자수 |
|--------|-------------|-------------|
| YouTube | 8-12자 | 15자 |
| 블로그 썸네일 | 10-15자 | 20자 |
| Instagram | 6-10자 | 12자 |
| Facebook | 8-12자 | 15자 |

**Readability Formula**:
```
가독성 점수 = (글자수 ÷ 10) × 명확성 × 대비도

최적: 8-12자 + 명확한 메시지 + 높은 대비

예시:
"월급 2배 벌기" → 7자 ✅
"직장인 부업으로 월급의 2배를 버는 확실한 방법" → 25자 ❌
```

### 2. Click-Triggering Phrases (클릭 유발 문구)

**고성능 트리거 단어 (High-Performance Triggers)**:

**호기심 트리거**:
```markdown
• "진짜 이유"
• "숨겨진 비밀"
• "아무도 안 알려주는"
• "충격적인 사실"
• "의외의 결과"

예시:
"유튜브 수익 안 나는 진짜 이유"
"아무도 안 알려주는 블로그 SEO"
```

**긴급성 트리거**:
```markdown
• "지금 바로"
• "오늘까지만"
• "놓치면 후회"
• "마감 임박"
• "한정"

예시:
"오늘까지만 공개"
"놓치면 후회할 정보"
```

**숫자 트리거**:
```markdown
• "TOP 5"
• "3가지 방법"
• "7일 만에"
• "100% 성공"
• "10분 컷"

예시:
"7일만에 10kg 감량"
"월 1000만원 버는 법 5가지"
```

**부정 트리거**:
```markdown
• "절대 하지마세요"
• "이것만은 피하세요"
• "망하는 이유"
• "실패하는 방법"
• "하면 안 되는"

예시:
"유튜브 절대 이렇게 하지마세요"
"블로그 망하는 이유 TOP 3"
```

**결과 트리거**:
```markdown
• "~한 결과"
• "Before & After"
• "~일 후기"
• "~만에 성공"
• "실제로 해봤더니"

예시:
"30일 실천한 결과 (충격)"
"3개월 사용 후기"
```

### 3. Visual Design Principles

**Color Psychology for Korean Market**:

| 색상 | 의미 | 적합한 주제 | CTR 영향 |
|------|------|-------------|----------|
| 빨강 (#FF0000) | 긴급, 중요, 열정 | 할인, 마감, 경고 | ⭐⭐⭐⭐⭐ |
| 노랑 (#FFFF00) | 주목, 밝음, 경쾌 | 팁, 긍정, 재미 | ⭐⭐⭐⭐⭐ |
| 파랑 (#0066FF) | 신뢰, 전문, 차분 | 교육, 비즈니스, 금융 | ⭐⭐⭐⭐ |
| 초록 (#00FF00) | 성장, 건강, 안전 | 건강, 재테크, 성공 | ⭐⭐⭐ |
| 보라 (#9900FF) | 프리미엄, 창의 | 고급, 아트, 영성 | ⭐⭐⭐ |
| 주황 (#FF6600) | 활력, 도전, 따뜻함 | 도전, 운동, 에너지 | ⭐⭐⭐⭐ |

**High-Contrast Combinations**:
```markdown
최고 조합 (CTR 30%+):

1. 노랑 배경 + 검정 글씨
   • 가독성: ⭐⭐⭐⭐⭐
   • 주목도: ⭐⭐⭐⭐⭐

2. 빨강 배경 + 흰색 글씨
   • 긴급성: ⭐⭐⭐⭐⭐
   • 주목도: ⭐⭐⭐⭐⭐

3. 파랑 배경 + 노랑 글씨
   • 대비: ⭐⭐⭐⭐⭐
   • 신뢰도: ⭐⭐⭐⭐

4. 검정 배경 + 빨강 글씨
   • 임팩트: ⭐⭐⭐⭐⭐
   • 고급감: ⭐⭐⭐⭐

피해야 할 조합:
❌ 빨강 배경 + 파랑 글씨 (눈 아픔)
❌ 초록 배경 + 노랑 글씨 (저급해 보임)
❌ 회색 배경 + 회색 글씨 (안 보임)
```

**Font Selection for Korean**:

```markdown
제목용 폰트 (굵고 임팩트 있게):

• 배민 주아체 (친근함, 귀여움)
  → 먹방, 일상 브이로그

• 프리텐다드 Bold (세련됨, 모던)
  → 비즈니스, 교육, IT

• 넥슨 Lv.2 Gothic Bold (가독성, 중립)
  → 리뷰, 정보, 뉴스

• 카페24 아네모네 (강렬함, 임팩트)
  → 챌린지, 운동, 도전

부제용 폰트 (얇고 보조적):

• 나눔고딕 Light
• 본고딕 Regular
• 스포카 한 sans Light

폰트 크기:
• 메인 텍스트: 80-120pt
• 서브 텍스트: 40-60pt
• 강조 단어: 100-150pt
```

**Layout Patterns**:

```markdown
Pattern 1: 중앙 정렬 (Center)
┌─────────────────┐
│                 │
│   메인 텍스트    │
│                 │
└─────────────────┘

사용처: 단순 메시지, 임팩트 중시
예시: "충격", "대박", "실화냐"

---

Pattern 2: 상단 + 하단 (Top & Bottom)
┌─────────────────┐
│   상단 텍스트    │
│                 │
│   (이미지 영역)  │
│                 │
│   하단 텍스트    │
└─────────────────┘

사용처: 대비, Before/After
예시:
상단: "Before"
하단: "After"

---

Pattern 3: 좌측 텍스트 + 우측 이미지
┌──────┬──────────┐
│ 메인 │          │
│ 텍스트│   이미지  │
│      │          │
└──────┴──────────┘

사용처: 리뷰, 제품 소개
예시: "이거 진짜 대박" + 제품 사진

---

Pattern 4: 대각선 배치 (Diagonal)
┌─────────────────┐
│ 텍스트\          │
│       \         │
│         \텍스트 │
└─────────────────┘

사용처: 역동적, 스포츠, 챌린지
예시: "30일 챌린지"
```

### 4. Emotion-Driven Copywriting

**감정별 최적 문구**:

**놀람 (Surprise)**:
```markdown
• "이게 진짜?"
• "실화냐"
• "충격"
• "말도 안 돼"
• "OMG"

예시 썸네일:
[큰 눈 이모지] "월급이 3배?? 실화냐"
```

**궁금증 (Curiosity)**:
```markdown
• "왜 그럴까?"
• "숨겨진 이유"
• "진실은?"
• "알고보니"
• "의외의"

예시 썸네일:
"유튜버가 안 알려주는 진짜 이유"
```

**공감 (Empathy)**:
```markdown
• "나만 그런가"
• "다들 이러시죠?"
• "공감 100%"
• "찐 후기"
• "솔직히"

예시 썸네일:
"나만 이렇게 못 하나요? (공감 ㅠㅠ)"
```

**긴급 (Urgency)**:
```markdown
• "지금 당장"
• "오늘까지만"
• "마지막 기회"
• "빨리"
• "서두르세요"

예시 썸네일:
"오늘까지만! 놓치면 후회"
```

**성취 (Achievement)**:
```markdown
• "성공"
• "해냈다"
• "달성"
• "~만에"
• "드디어"

예시 썸네일:
"7일만에 10kg 감량 성공!"
```

### 5. Platform-Specific Guidelines

**YouTube 썸네일**:
```markdown
크기: 1280x720px (16:9 비율)
안전 영역: 좌우 5% 여백 (TV 잘림 방지)

텍스트 최적화:
• 8-12자 이내
• 모바일에서도 읽힐 크기 (80pt+)
• 얼굴 위에 텍스트 배치 지양

색상 전략:
• YouTube 빨강(#FF0000)과 대비되는 색
• 노랑, 파랑, 초록 권장
• 화이트/블랙 테두리로 가독성 ↑

검증 방법:
1. 썸네일 축소해서 보기 (스마트폰 크기)
2. 3초 안에 읽히는지 테스트
3. 연관 동영상에서 눈에 띄는지 확인
```

**네이버 블로그 썸네일**:
```markdown
크기: 800x800px (1:1 비율) 또는 800x450px (16:9)
파일 크기: 200KB 이하 (로딩 속도)

텍스트 최적화:
• 10-15자 이내
• 키워드 필수 포함
• 상단에 배치 (미리보기에서 보임)

색상 전략:
• 네이버 그린(#03C75A)과 대비
• 밝은 배경색 선호 (신뢰감)
• 파스텔톤 효과적

SEO 고려:
• 파일명: "키워드-썸네일.jpg"
• Alt 텍스트 작성
• 워터마크 (블로그 URL)
```

**Instagram 썸네일**:
```markdown
크기: 1080x1080px (1:1) 또는 1080x1350px (4:5)

텍스트 최적화:
• 6-10자 이내 (더 짧게!)
• 감성적 표현
• 이모지 활용

색상 전략:
• 통일된 브랜드 컬러
• 피드 전체 조화
• 파스텔 or 비비드 (일관성)

디자인 팁:
• 여백 충분히 (미니멀)
• 감성적 폰트
• 필터 일관성 유지
```

## Examples

### Example 1: YouTube 재테크 채널

**상황**: 주식 투자 초보자 타겟, CTR 2% → 12% 개선

**Before (CTR 2%)**:
```
[썸네일]
배경: 단순 파랑
텍스트: "주식 투자 시작하는 방법과 초보자를 위한 팁"
색상: 흰색 글씨
폰트: 기본 폰트

문제점:
❌ 텍스트 너무 김 (20자)
❌ 임팩트 부족
❌ 호기심 유발 안 됨
❌ 모바일에서 안 읽힘
```

**After (CTR 12%)**:
```
[썸네일]
┌─────────────────────────┐
│                         │
│    주식 3일 만에       │
│    1000만원?!          │
│                         │
│   [충격받은 얼굴 사진]   │
│                         │
│    💰 실제 후기 💰     │
└─────────────────────────┘

디자인 요소:
• 배경: 노랑→빨강 그라데이션
• 메인 텍스트: "주식 3일 만에" (검정, 굵게)
• 강조 텍스트: "1000만원?!" (빨강, 더 굵게, 150pt)
• 서브 텍스트: "💰 실제 후기 💰" (흰색)
• 폰트: 프리텐다드 Extra Bold
• 테두리: 검정 외곽선 (가독성)

개선 요소:
✅ 9자로 축약
✅ 숫자로 구체성 ↑
✅ 물음표로 호기심 유발
✅ 이모지로 주목도 ↑
✅ 대비 강화 (모바일 가독성)
```

### Example 2: 블로그 요리 레시피

**상황**: 검색 유입 증가 목표

**Before**:
```
[썸네일]
배경: 음식 사진
텍스트: "초코칩 쿠키"
색상: 흰색
폰트: 가는 폰트

문제점:
❌ 차별화 없음
❌ 클릭 동기 부족
❌ 검색 최적화 미흡
```

**After**:
```
[썸네일 - 상단]
┌─────────────────────────┐
│  실패 없는 초코칩 쿠키   │  ← 빨강 박스
│                         │
│   [쿠키 클로즈업 사진]   │
│                         │
│     10분 완성!          │  ← 노랑 원형
│     초보자 가능 ✅      │
└─────────────────────────┘

디자인 요소:
• 상단 박스: 빨강 배경 + 흰색 글씨
• "실패 없는" 강조 (두껍게)
• 하단 원형: 노랑 배경 + 검정 글씨
• "10분" 숫자 강조
• 체크마크로 신뢰감

개선 요소:
✅ "실패 없는" → 안심감 제공
✅ "10분" → 간편함 강조
✅ "초보자 가능" → 타겟 명확화
✅ 키워드 포함 (SEO)
```

개선 요소:
✅ 감성적 색감

✅ 브랜드 통일성
✅ 참여 유도 요소

## Common Patterns

### Pattern 1: Thumbnail Template Library

```markdown
템플릿 1: 숫자 강조형
용도: 리스트, 팁, 순위
구조:
- 배경: 단색 or 그라데이션
- 숫자: 초대형 (200pt)
- 텍스트: 8자 이내
- 색상: 노랑+검정 or 빨강+흰색

예시:
"TOP 5", "7가지 방법", "3분 완성"

---

템플릿 2: Before/After형
용도: 변화, 리뷰, 후기
구조:
- 좌우 분할
- Before: 어두운 색 + 문제 상황
- After: 밝은 색 + 해결 상황
- 화살표 (→) 강조

예시:
"30일 후 변화 (충격)"

---

템플릿 3: 질문형
용도: 궁금증 유발, 공감
구조:
- 물음표 크게
- 공감 가는 질문
- 얼굴 표정 (고민하는)
- 파스텔 배경

예시:
"나만 이런가요?", "왜 안 될까?"

---

템플릿 4: 긴급형
용도: 할인, 마감, 이벤트
구조:
- 빨강 배경
- "오늘까지", "마감" 등
- 타이머 or 시계 이미지
- 굵은 글씨

예시:
"오늘까지만!", "24시간 남음"
```

### Pattern 2: A/B Testing Checklist

```markdown
동일 영상, 4가지 썸네일 테스트:

Version A: 숫자 강조
"월 1000만원 버는 법"

Version B: 질문형
"월급 2배? 어떻게?"

Version C: 부정형
"절대 이렇게 하지마세요"

Version D: 결과형
"30일 실천 결과 (충격)"

테스트 기간: 각 버전 3일씩
측정 지표: CTR, 평균 시청 시간
최종 선택: 가장 높은 CTR 버전

실제 결과 예시:
Version A: CTR 8.2%
Version B: CTR 11.5% ← 최종 선택
Version C: CTR 6.8%
Version D: CTR 9.1%
```

## Best Practices

### ✅ DO

**디자인 원칙**:
- **대비 극대화**: 배경과 텍스트 색상 확실히 구분
- **여백 확보**: 텍스트 주변 최소 10% 여백
- **일관성 유지**: 채널/블로그 전체 통일감
- **모바일 우선**: 작은 화면에서 테스트

**텍스트 작성**:
- **짧고 강렬하게**: 8-12자 목표
- **숫자 활용**: 구체적일수록 클릭률 ↑
- **감정 자극**: 놀람, 호기심, 공감
- **키워드 포함**: SEO 고려 (블로그)

**색상 선택**:
- **3색 이내**: 너무 많으면 산만
- **브랜드 컬러**: 일관된 색상 팔레트
- **계절감 반영**: 시즌별 색감 조정
- **경쟁사 분석**: 같은 주제 상위 썸네일 연구

### ❌ DON'T

**절대 금지**:
- **클릭베이트**: 내용과 다른 썸네일 (신뢰도 ↓)
- **텍스트 과다**: 15자 초과 (안 읽힘)
- **저해상도 이미지**: 흐릿하면 비전문적
- **복잡한 디자인**: 3초 안에 이해 안 되면 실패
- **저작권 침해**: 타인 이미지 무단 사용

**피해야 할 실수**:
- 너무 많은 색 사용 (5색 이상)
- 가는 폰트 (모바일에서 안 보임)
- 중요한 텍스트를 모서리에 배치
- 배경과 텍스트 색이 비슷함
- 일관성 없는 스타일 (매번 다름)

## Troubleshooting

### Issue: "CTR이 너무 낮아요 (2% 이하)"

**진단 체크리스트**:
```markdown
□ 텍스트가 15자 이상인가? → 10자로 줄이기
□ 모바일에서 읽히나? → 폰트 크기 ↑
□ 호기심을 유발하나? → 트리거 단어 추가
□ 경쟁 영상과 비교했나? → 차별화 요소 추가
□ 색상 대비가 충분한가? → 명암 조절

개선 전후 예시:
Before: "요리 레시피 소개" (CTR 1.8%)
After: "10분 완성! 실패 없는 요리" (CTR 8.3%)
```

### Issue: "썸네일은 좋은데 내용이 실망스럽다는 댓글"

**원인**: 클릭베이트 (내용 ≠ 썸네일)

**Solution**:
```markdown
원칙: 썸네일은 영상의 '하이라이트'이지 '과장'이 아님

잘못된 예:
영상: 주식 투자 기초 설명
썸네일: "3일만에 1억 벌기" ← 과장

올바른 예:
영상: 주식 투자 기초 설명
썸네일: "주식 초보 탈출 3단계" ← 정확

체크 방법:
1. 썸네일 문구가 영상에 실제로 나오는가?
2. 약속한 내용을 영상에서 전달하는가?
3. 과장된 표현은 없는가?
```

### Issue: "디자인 실력이 없어요"

**Solution: 초보자용 도구 활용**

```markdown
추천 도구:

1. 미리캔버스 (무료, 한글)
   • 썸네일 템플릿 다양
   • 한글 폰트 풍부
   • 클릭 몇 번으로 완성

2. Canva (유/무료, 영어)
   • 전문가 수준 템플릿
   • AI 기능 (배경 제거 등)
   • 협업 가능

3. 포토스케이프 X (무료)
   • 간단한 편집
   • 텍스트 효과
   • 초보자 친화적

빠른 시작 가이드:
Step 1: 템플릿 선택
Step 2: 배경 이미지 교체
Step 3: 텍스트만 수정 (8-12자)
Step 4: 색상 조정 (브랜드 컬러)
Step 5: 다운로드 (JPG, 1280x720)

시간: 10분 이내 완성 가능
```
